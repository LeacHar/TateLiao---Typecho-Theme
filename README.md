# TateLiao---Typecho-Theme

一款TypechoTheme主题，基于BootStrap等前端框架制作，演示地址：https://www.tateliao.me/

# 开发环境

Apache + PHP(4.5) + MySql(5.5) + Typecho(1.1)

# 功能描述

1. 自适应优化
2. 图片懒加载
3. instantclick快速响应
4. pjax无刷新跳转
5. 导航栏跟随优化
6. 代码高亮
7. 图片灯箱
8. 丰富的主题设置

# 更新说明

以后更新将会实现以下功能
1. 主题选项增加头部，底部自定义内容
2. 评论无刷新提交
